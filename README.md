# ELB Construction Website

Single-page React website for **ELB - Entreprise Leader du Batiment**.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Notes

- Primary language: French
- Contact flow: WhatsApp/call CTA only (no backend form submission in V1)
- Main brand assets are in `public/assets/`
